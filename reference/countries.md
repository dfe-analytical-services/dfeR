# Lookup for valid country names and codes

A lookup of ONS geography country names and codes, as well as some
custom DfE names and codes. This is used as the definitive list for the
screening of open data before it is published by the DfE.

## Usage

``` r
countries
```

## Format

### `countries`

A data frame with 10 rows and 2 columns:

- country_name:

  Country name

- country_code:

  Country code

## Source

curated by explore.statistics@education.gov.uk, ONS codes sourced from
https://geoportal.statistics.gov.uk/search?q=countries%20names%20and%20codes
