# Potential names for geography and time columns

Potential names for geography and time columns in line with the ones
used for the explore education statistics data screener.

## Usage

``` r
geog_time_identifiers
```

## Format

### `geog_time_identifiers`

A character vector with 38 potential column names in snake case format.

## Source

curated by explore.statistics@education.gov.uk. [Guidance on time and
geography data.](https://www.shorturl.at/j4532)
